# AutomaticAtlas

一个 **Unity 自动图集工具**。

支持根据 **预制体引用关系** 或 **文件夹目录结构** 自动生成 SpriteAtlas，并以简单直接的方式合并重复资源。

---

## 🚀 功能入口

Unity 顶部菜单：Tools -> AutomaticAtlas



---

## ⚙️ 初次使用

第一次使用前，需要先创建配置资产 `AtlasConfig`。

### 创建方式

在 Unity 项目视图中任意位置：右键 → Create → Create AtlasConfig



创建完成后，在 Inspector 中配置相关参数即可开始使用。
